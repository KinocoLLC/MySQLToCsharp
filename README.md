## MySQLToCsharp


## Generate MySQL Lexer/Parser/Listener/Visitor from ANTLR4 grammer

run script to generate C# class files.
it calls `docker-compose up` and generate lexer, parser, listener and visitor class.

```
# windows
gen.bat

# macos/linux
gen.sh
```
